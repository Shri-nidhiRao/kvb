Complete Indoor Farming Solution

The Smart Vertical Microgreens Growing System is a compact, sensor‑driven indoor farming solution designed to enable high‑quality, pesticide‑free microgreens production in a fully controlled environment. Unlike conventional systems that focus only on lighting or hydroponics, this system integrates production, environmental control, automation, and performance evaluation into a single unit – making it suitable for urban farming, research, and commercial applications. This approach addresses a key gap identified across multiple research studies, where no single system integrates growth, control, and post‑harvest considerations together.

Smart Automation Logic

Mode / Feature Description
Temperature Control System monitors continuously; if temperature exceeds set limit → cooling fan ON, else OFF.
Lighting Control Automated timer‑based lighting cycle (e.g., 18h OFF / 6h ON, configurable).
Irrigation – Manual Mode User‑controlled ON/OFF.
Irrigation – Timer Mode Irrigation at fixed intervals.
Irrigation – Smart Mode Checks soil moisture periodically; if below threshold → pump ON, valve opens, water sprayed; auto OFF after set duration.
